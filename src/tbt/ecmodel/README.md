# github-maps-analytics-error-classification
Repository for training, serving and monitoring error classification models. 
The repository can be imported to other projects as a submodule. 

## How to Add This Submodule to Your Project

In order to include this submodule in your project, follow the steps below.

1. Navigate to your project in terminal.

2. Add the submodule to your repository with the following command:

   ```bash
   git submodule add https://github.com/tomtom-internal/github-maps-analytics-error-classification.git
   ```

3. Initialize and clone the new submodule:

   ```bash
   git submodule sync && git submodule update --init --recursive
   ```
